# Compze.DocumentDb.Sqlite

SQLite backend for [Compze.DocumentDb](https://www.nuget.org/packages/Compze.DocumentDb) — a SQLite `IDocumentDbSqlLayer` implementation providing typed document storage on SQLite.

## What is Compze?

Compze is a .NET framework for building expressive domains through **Teventive programming** and **Typermedia APIs**. [Learn more](https://compze.net/)

## Installation

```shell
dotnet add package Compze.DocumentDb.Sqlite
```
