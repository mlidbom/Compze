# Compze.DependencyInjection.Autofac.Extensions.Hosting

Autofac hosting integration for the Compze framework. Bridges a Compze `AutofacDependencyInjectionContainer` into `IHostBuilder` via a custom `IServiceProviderFactory<ContainerBuilder>`.
