# Compze.DocumentDb.MicrosoftSql

SQL Server backend for [Compze.DocumentDb](https://www.nuget.org/packages/Compze.DocumentDb) — a SQL Server `IDocumentDbSqlLayer` implementation providing typed document storage on SQL Server.

## What is Compze?

Compze is a .NET framework for building expressive domains through **Teventive programming** and **Typermedia APIs**. [Learn more](https://compze.net/)

## Installation

```shell
dotnet add package Compze.DocumentDb.MicrosoftSql
```
