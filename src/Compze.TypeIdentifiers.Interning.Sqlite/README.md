# Compze.TypeIdentifiers.Interning.Sqlite

SQLite backend for [Compze.TypeIdentifiers.Interning](https://www.nuget.org/packages/Compze.TypeIdentifiers.Interning) — a SQLite `ITypeIdInternerPersistence` implementation that stores the database-local `TypeId` → `int` mapping in SQLite.

## What is Compze?

Compze is a .NET framework for building expressive domains through **Teventive programming** and **Typermedia APIs**. [Learn more](https://compze.net/)

## Installation

```shell
dotnet add package Compze.TypeIdentifiers.Interning.Sqlite
```
