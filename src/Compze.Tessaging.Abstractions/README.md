# Compze.Tessaging.Abstractions

Handler registration contracts for the [Compze](https://github.com/mlidbom/Compze) framework.

## What is Compze?

Compze is a .NET framework for building expressive domains through **Teventive programming** and **Typermedia APIs**. [Learn more](https://compze.net/)

## What's in this package?

- **`ITessageHandlerRegistrar`** ‚Äî interface for registering handlers for tevents, tommands, and tueries
- **`TessageHandlerRegistrarWithDependencyInjectionSupport`** ‚Äî handler registration with automatic dependency resolution

## Installation

```shell
dotnet add package Compze.Tessaging.Abstractions
```
