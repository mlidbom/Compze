# Compze.DocumentDb.PostgreSql

PostgreSQL backend for [Compze.DocumentDb](https://www.nuget.org/packages/Compze.DocumentDb) — a PostgreSQL `IDocumentDbSqlLayer` implementation providing typed document storage on PostgreSQL.

## What is Compze?

Compze is a .NET framework for building expressive domains through **Teventive programming** and **Typermedia APIs**. [Learn more](https://compze.net/)

## Installation

```shell
dotnet add package Compze.DocumentDb.PostgreSql
```
