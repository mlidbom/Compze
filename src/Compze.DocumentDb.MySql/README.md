# Compze.DocumentDb.MySql

MySQL backend for [Compze.DocumentDb](https://www.nuget.org/packages/Compze.DocumentDb) — a MySQL `IDocumentDbSqlLayer` implementation providing typed document storage on MySQL.

## What is Compze?

Compze is a .NET framework for building expressive domains through **Teventive programming** and **Typermedia APIs**. [Learn more](https://compze.net/)

## Installation

```shell
dotnet add package Compze.DocumentDb.MySql
```
