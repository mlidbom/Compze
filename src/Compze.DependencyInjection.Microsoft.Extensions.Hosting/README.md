# Compze.DependencyInjection.Microsoft.Extensions.Hosting

Microsoft DI hosting integration for the Compze framework. Bridges a Compze `MicrosoftDependencyInjectionContainer` into `IHostBuilder` via a custom `IServiceProviderFactory<IServiceCollection>`.
