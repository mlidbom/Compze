ForGenericEvent<THandledEvent>(Action<THandledEvent>)

Lets you register handlers for event interfaces that may be defined outside of the event hierarchy you specify with TEvent. Useful for listening to generic events such as IAggregateCreatedEvent or IAggregateDeletedEvent Be aware that the concrete event received MUST still actually inherit TEvent or there will be an InvalidCastException
