MessageTypes.Remotable.AtMostOnce.AtMostOnceHypermediaCommand.DeduplicationIdHandling

How and when a MessageId is generated is vital to correctly maintain the At most once delivery guarantee. When creating the command in the backend service we must generate a new MessageId. When binding the command in a UI we must reuse the MessageId generated by the backend. This enum helps make that important distinction explicit in your code.
